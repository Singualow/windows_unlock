# Android 蓝牙钥匙

Android 应用面向 Android 15 及以上设备，包名为
`com.singu.proximityunlock`。当前界面采用与 Windows 控制中心一致的浅色视觉语言，
但按手机单手操作重新组织信息。

## 界面结构

- 顶部状态卡：BLE 前台服务、当前模式和配对状态，以及启动/停止和配对入口。
- 解锁模式：安全模式与便捷模式使用两个等高选项；手机端模式需与电脑端保持一致。
- 配对设备：显示不可导出密钥的 StrongBox/TEE 后端，可重新配对或撤销电脑。
- 安全日志：显示最近 5 条脱敏事件，本机最多保存 16 条，不记录设备标识或密码学材料。
- 配对面板：扫描电脑二维码后通过深链接自动填入，也支持粘贴
  `proximityunlock://pair` 链接。

## 安全操作

切换模式和撤销设备要求手机处于解锁状态；启用便捷模式前还会显示风险确认。
停止蓝牙钥匙不会删除配对，之后可直接点击“启动蓝牙钥匙”恢复。撤销设备会删除
配对、presence key 和 Android Keystore 中的两把签名密钥。

最终用户应安装 Release 中使用个人证书签名的 `ProximityUnlock-Android.apk`，覆盖
升级会保留现有配对和不可导出密钥。
