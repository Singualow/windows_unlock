# 蓝牙解锁桌面控制中心

桌面控制中心采用 Tauri 2、React、TypeScript 和 Vite。它只替换当前用户会话中的
可视化界面，不修改 Go 服务、C++ 凭据提供程序或 Android 蓝牙协议。

## 界面功能

- 概览：连接状态、实时 RSSI 趋势、六个独立解锁选项和脱敏安全事件。
- 设备：程序内生成配对二维码、撤销手机、手动修改解锁/锁定阈值和距离校准。
- 日志：展示本次服务运行最近 100 条认证、授权、凭据提交和脱敏失败原因；连续重复的
  认证成功或同类认证失败只记录一次，不显示秘密数据。
- 设置：切换安全/便捷模式、自动锁定、高灵敏模式、RSSI 趋势预测、锁屏后立即解锁、认证失败冷却、临时暂停，以及凭据提供程序启停、Windows 密码更新和完整卸载。
- 托盘：关闭主窗口后继续运行，可重新显示界面、立即锁定电脑或退出界面。
- 后台：正式版采用 Windows GUI 子系统，不创建控制台窗口；开机启动时隐藏主窗口并保持单实例运行。

概览页的六个解锁选项使用同一个等高行组件；设置页额外显示“认证失败冷却”，七个
选项同样等高。便捷模式、高灵敏模式、趋势预测、锁屏后立即解锁和关闭失败冷却都会
先显示风险确认。高灵敏和趋势预测的参数分别位于独立弹窗，不与普通阈值混排。

高灵敏模式使用独立的 RSSI 触发阈值，不覆盖普通模式的解锁/锁定阈值；锁定保护线
由触发阈值自动减去 `8 dB` 得到。锁屏时首个符合阈值的新鲜广播会在约 `0.2` 秒内
发起认证，陈旧 GATT 交换最多占用约 `1.5` 秒；启用或修改阈值时会重新开始一个
短暂的有效证明宽限期，避免使用切换前的陈旧状态立即锁定电脑。首次认证失败后仍会
持续重试；只有连续 `10` 秒没有恢复认证才允许自动锁定，任意一次成功认证会重置计时。

“多普勒预测”是面向体验的 RSSI 趋势预测：服务对最近约 `3.5` 秒的样本做线性趋势
估计，按 `1` 至 `100` 的灵敏度决定预测窗口与最小增强速度。预测命中只会提前发起
挑战，不会绕过手机签名；预测解锁后连续 `10` 秒没有新鲜有效证明会重新锁定，且这条
预测专用安全复核不受普通“自动锁定”开关影响。

概览页的距离趋势使用真实的滚动 `10` 分钟时间轴，每 `2` 秒写入一个采样点。程序刚
启动时，已有数据只占据时间轴右侧的实际时长，不会被拉伸成完整十分钟。

## 安全接口

Rust 后端只允许调用预先列出的控制操作，通过
`\\.\pipe\ProximityUnlock.Control.v1` 与现有 Go 服务通信。前端不能直接访问
配置文件、LSA 私密数据、凭据提供程序授权管道或系统密码。

浏览器预览使用不包含秘密的本地演示状态；编译后的 Tauri 程序始终读取真实服务。

## 开发构建

以下内容仅供项目开发者使用，最终用户仍通过图形安装程序安装，不需要执行命令。

需要 Node.js、Rust/Cargo、WebView2 和现有 Windows 构建环境。进入 `desktop` 后：

```powershell
npm install
npm run build
npm run tauri build -- --no-bundle
```

生成的开发阶段可执行文件位于
`desktop\src-tauri\target\release\proximity-unlock-desktop.exe`。项目构建脚本会将其
重命名为 `ProximityUnlock.exe`，内嵌到 Windows 单文件图形安装程序中；最终用户
只需双击安装程序，不需要运行脚本。
